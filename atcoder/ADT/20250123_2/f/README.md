# 解説
前後から文字列をチェックして、各条件を満たすかを判定します。

## TはSと等しい
長さも同一文字数も同じ場合です。

## S が1文字多い
S-1=Tで、同一文字数がTの文字数(Tの部分は全て満たしている)の場合です。

## T が1文字多い
T-1=Sで、同一文字数がSの文字数(Sの部分は全て満たしている)の場合です。

## 1文字置換
SとTの文字数が同じで、同一文字が1文字以外の場合です。
