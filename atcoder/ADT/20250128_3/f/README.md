# 解説
累積和を使います。

最初にMまでのスコアの最大値を求めます。

そこを基準に先頭と末尾を1つずつずらして、スコアを計算します。

最終的な最大スコアを出力します。

## 累積和の使い所

`5*1+4*2+-1*3` が

`____4*1+-1*2+8*3`になるときのことを考えます。

`8*3`の部分は`m * a[m + i]`で単純に計算できますが、
引く部分は計算が簡単にできません。

ここで累積和を使います。
`5*1+4*1`の分が差し引かれていることがわかります。`i1*1+i2*1+i3*1+...`のように一般化できます。
そのため累積和を使って、差し引かれる部分の取得ができます。
